### Context
A next-generation wearable required a mobile application that could sync biometric data perfectly across the edge network without hitting a centralized cloud server.

### Technical Implementation
* Constructed entirely in **React Native** (New Architecture).
* Built custom JNI (Android) and Objective-C (iOS) bridges.
* Embedded a local SQLite vector store for offline processing.
